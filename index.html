When you click go you'll be asked for a game code and player number, both should be agreed with your opponent before you start.


If you're player 1 then start with a piece in the bottom-left corner, if you're player 2 then top-right.


To place a piece click on a piece on the left, rotate or flip using the buttons and then click a cell on the board to place.


If you load the game and the board is not blank, please choose a new code and click "New game".


Also, don't quit the tab mid-game. Going back into it doesn't work properly yet, it's a known bug. sorry
